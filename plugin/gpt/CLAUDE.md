## GPT Plugin for Gobot

### API documentation

You can append "`.md`" to the end of OpenAI API documentation links (minus query strings) to get the markdown version of the documentation.
Normal links are blocked with 403.

- Responses API Overview: <https://developers.openai.com/api/reference/responses/overview.md> (follow the links inside)
- Create Response API
  reference: <https://developers.openai.com/api/reference/resources/responses/methods/create/index.md>
- Image Vision (Analyze Images): <https://developers.openai.com/api/docs/guides/images-vision.md>
- Reasoning/Thinking: <https://developers.openai.com/api/docs/guides/reasoning.md>
- Tool Use: <https://developers.openai.com/api/docs/guides/tools.md>
- Function Calling: <https://developers.openai.com/api/docs/guides/function-calling.md>
- Managing Conversation State: <https://developers.openai.com/api/docs/guides/conversation-state.md>
- Brave Search: <https://api-dashboard.search.brave.com/api-reference/web/search/get>

Tool call errors must be written in English.
